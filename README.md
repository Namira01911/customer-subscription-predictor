# customer-subscription-predictor
Problem Statement
Businesses face the problem of identifying their potential customers who may subscribe to their services. Predicting their subscription intentions will help businesses optimize their marketing expenses.

🎯 Objective
Create a simple ML-based web prototype that predicts customer subscription based on:
Age
Purchase Amount
🛠 Tech Stack
Python
Streamlit – Front End
Basic Machine Learning Logic – Threshold-based approach
Git & GitHub
🧠 Approach
User will input their Age and Purchase Amount
System will run the prediction logic
If conditions met:
"Customer Will Subscribe"
Else:
"Customer Will NOT Subscribe"
🏗 Architecture
User Input – Streamlit Front End
↓
Prediction Logic – Python Back End
↓
Result will be displayed on the Web App
🤖 ML + GenAI Integration
Machine Learning – Basic Machine Learning Logic – Threshold-based approach
Generation of Artificial Intelligence – Used ChatGPT for Architecture Development, Debugging, and Improvement Suggestions
⚖ Ethical Considerations
May be biased due to Basic Machine Learning Logic
Requires large unbiased data sets
Not to be used for discriminatory marketing
💼 Business Feasibility
Helps businesses optimize their ad targeting
Helps businesses save money by reducing marketing expenses
Can be integrated with existing CRM Systems
